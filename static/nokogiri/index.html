Nokogiri

23 July 2017 on Ruby on Rails, Nokogiri
用途

解析Html結構, 可用於爬蟲

基本方法
  1. 讀取Html
    doc = Nokogiri::HTML(file, nil, "編碼")
    file 可以是是網址 
require 'open-uri'  
doc = Nokogiri::HTML(open( url ), nil, 'UTF-8' )

或是檔案路徑 

doc = Nokogiri::HTML(File.open("index.html"), nil, 'UTF-8' )

或是直接的字串 

doc = Nokogiri::HTML("<html><h1>Hello World!!</h1></html>", nil, 'UTF-8' )
  1. 選擇器
    • xpath
      doc.xpath("h1")
    • css
      doc.css(".class_name")
    • 選擇全部
      doc.css("//.class_name")
  2. 取值
    • text 該元素內的文字(會自動去除html)
    • to_html 包含該元素的完整html
    • inner_html 該元素內的完整html
    • [attr] 該元素之屬性
參考

https://wwssllabcd.github.io/blog/2012/10/25/how-to-use-nokogiri/
