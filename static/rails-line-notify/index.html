Rails Line notify 串接

01 October 2017 on Ruby on Rails, Line, Line Notify
line notify 網站設定
  • 註冊 line notify
    https://notify-bot.line.me/zh_TW/
  • 管理登錄服務
  • 新建登錄服務
  • 設定 Callback URL (最多五組)
    如果使用 gem 'omniauth-line-notify' 設定如下
http://DOMAIN/auth/line_notify/callback
  • 取得 LINE_NOTIFY_CLIEDNT_ID + LINE_NOTIFY_APP_SECRET
Rails 設定
  • add gem 'omniauth-line-notify in Gemfile
  • in config/initializers/omniauth.rb
Rails.application.config.middleware.use OmniAuth::Builder do  
    provider :line_notify, ENV['LINE_NOTIFY_CLIEDNT_ID'], ENV['LINE_NOTIFY_APP_SECRET'], scope: 'notify'

  OmniAuth.config.on_failure = Proc.new { |env|
    OmniAuth::FailureEndpoint.new(env).redirect_to_failure
  }
end
  • in config/routes
get'/auth/line_notify/callback' , to: "users/omniauth_callbacks#line_notify"
  • in omniauthcallbackscontroller.rb 存取使用者授權後的linenotifytoken
發送訊息
RestClient.post 'https://notify-api.line.me/api/notify', {message: @line_notify_message}, { :Authorization => "Bearer #{@line_notify_token}"}

# @line_notify_message => 訊息
# @line_notify_token => 取得的token
參考
  • https://github.com/blueplanet/omniauth-line-notify (但他範例中的provider是錯誤的)
