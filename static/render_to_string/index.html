Ruby on Rails method render_to_string

22 July 2017 on Ruby on Rails
來源

https://apidock.com/rails/ActionController/Base/rendertostring

用法

基本上跟在view中render是一樣的, 只差別是在controller中使用
* render partial
render_to_string(partial: "/users/show", locals: {user: user})
* render template
render_to_string(template: "/users/show", locals: {user: user})

如果要在其他地方使用(例如自定義的class或是model)則必須ApplicationController.new.render_to_string()

另外如果render的view中有用到urlhelper的東西, 遇到
ActionView::Template::Error (undefined method 'host' for nil:NilClass) 這個錯誤則在
/config/initializers/applicationcontroller_renderer.rb 加入
include Rails.application.routes.url_helpers
include ApplicationHelper

def default_url_options {host: Rails.application.routes.default_url_options[:host]} end
