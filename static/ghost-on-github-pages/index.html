buster

用途打包文章成靜態檔案才能放到github pages上
安裝指令 

pip install buster
部署步驟
buster generate  
git add .  
git commit -am "add post"  
git push origin master
Jquery not working on github pages

因為預設版型內的jquery cdn 不是 https, 所以在github pagaes 上不能讀取. 所以找到ghost裡面 content/theme/casper/default.hbs 幫jquery路徑修改為https版本即可.
