Rails Cache

25 July 2017 on Ruby on Rails, cache
設定

install memchached 

# mac
brew install memcached  
# ubuntu
$ sudo apt-get install memcached

in Gemfile 

# cache
gem "dalli"

in config/environments/production.rb 修改cache_store方式 

- config.cache_store = :memory_store
+ config.cache_store = :mem_cache_store

development 下測試, 開起cache 

config.action_controller.perform_caching = true

啟用 memcached

* memcached -l 127.0.0.1 -P 11211 -m 128 -d for deamon
* memcached -l 127.0.0.1 -P 11211 -m 128 -vv for development debug
使用

https://ihower.tw/rails/caching.html
